# Heart Disease Analysis Dashboard

## Overview

This project focuses on analyzing heart disease data using modern data analytics and visualization techniques. The aim of the project is to explore patient health data and identify patterns that may indicate the presence of heart disease. By using data visualization tools, complex medical data can be transformed into clear and meaningful visual insights. In this project, Tableau is used to create interactive charts, dashboards, and stories that help analyze the dataset effectively. The visualizations allow users to understand how different medical attributes such as age, cholesterol level, blood pressure, and heart rate relate to heart disease.

The project also demonstrates how data visualization results can be integrated into a web application. A Flask-based web interface is developed to display the Tableau dashboard directly in a browser. This integration allows users to access the visual analysis easily and interact with the dashboard without opening Tableau separately. The combination of data visualization and web integration helps present healthcare data in a structured and accessible format.

## Introduction

Heart disease is one of the leading causes of death across the world and remains a major public health concern. Early identification of risk factors is essential for prevention and effective treatment. Hospitals and healthcare organizations collect large amounts of patient data, including information about age, cholesterol levels, blood pressure, heart rate, and other medical conditions. However, raw data alone does not provide meaningful insights unless it is analyzed properly using data analytics tools.

Data visualization plays an important role in understanding medical datasets by transforming numerical data into graphical representations. Visual tools help doctors, researchers, and analysts identify trends, relationships, and risk factors more easily. This project uses Tableau to analyze heart disease data and present it through interactive visualizations and dashboards. By studying these visual patterns, users can gain insights into how different health indicators contribute to heart disease.

In addition to visualization, the project integrates the dashboard into a web application using the Flask framework. This allows the analysis to be accessed through a web interface, making it easier for users to view and interact with the results. The project demonstrates how data analytics, visualization tools, and web technologies can work together to present healthcare insights in an effective and user-friendly manner.

## Problem Statement

Medical datasets contain valuable information that can help understand the factors responsible for heart disease. However, raw data alone is difficult to interpret. The aim of this project is to analyze patient data and visualize important health indicators that may influence the occurrence of heart disease.

## Dataset Description

The dataset used in this project contains medical attributes related to heart disease patients. The attributes include age, gender, chest pain type, cholesterol level, blood pressure, fasting blood sugar, and maximum heart rate. These variables help analyze how different health conditions are associated with heart disease.

## Data Visualization

Data visualization is performed using Tableau. Different charts and graphs are created to analyze the dataset and identify patterns. These visualizations make complex medical data easier to understand and help highlight important relationships between variables.

## Dashboard

The Tableau dashboard combines multiple visualizations into a single interface. It allows users to explore the dataset interactively and understand trends in heart disease data. The dashboard provides a clear visual summary of the analysis.

## Web Integration

The Tableau dashboard is integrated into a web application using Flask. Flask is a lightweight Python framework used to build web applications. The dashboard is embedded into the web page using an HTML iframe, allowing users to view and interact with the visualization through a browser.

## Technologies Used

The project uses Python and Flask for web integration, Tableau for data visualization, and HTML for building the web interface. GitHub is used to manage and store the project files.

## Conclusion

This project demonstrates how data analytics and visualization can help analyze healthcare datasets. By combining Tableau dashboards with a Flask web application, the project provides an interactive way to explore heart disease data and gain meaningful insights.
